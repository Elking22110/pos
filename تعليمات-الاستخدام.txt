1. متطلبات النظام
========================================

الحد الأدنى:
- Windows 10 أو أحدث
- 4 GB RAM
- 2 GB مساحة فارغة
- اتصال بالإنترنت (للتحديثات)

الموصى به:
- Windows 11
- 8 GB RAM أو أكثر
- 5 GB مساحة فارغة
- اتصال سريع بالإنترنت

========================================
2. التثبيت والإعداد
========================================

أولاً: تثبيت Node.js
1. اذهب إلى https://nodejs.org
2. حمل أحدث إصدار LTS
3. ثبت Node.js مع npm
4. تأكد من التثبيت: node --version

ثانياً: تثبيت التبعيات
1. افتح Command Prompt في مجلد المشروع
2. اكتب: npm install
3. انتظر حتى انتهاء التثبيت

========================================
3. تشغيل التطبيق
========================================

الطريقة الأولى: تشغيل في المتصفح
- انقر مرتين على: scripts\build\start-web.bat
- سيتم فتح التطبيق في المتصفح على http://localhost:5173

الطريقة الثانية: تشغيل كتطبيق سطح مكتب (وضع التطوير)
- انقر مرتين على: scripts\build\start-electron-fixed.bat
- سيتم فتح التطبيق كتطبيق سطح مكتب مع أدوات المطور

الطريقة الثالثة: تشغيل كتطبيق سطح مكتب (وضع الإنتاج)
1. انقر مرتين على: scripts\build\build-app.bat
2. انتظر حتى انتهاء البناء
3. انقر مرتين على: scripts\build\run-app.bat

========================================
4. بناء التطبيق للتوزيع
========================================

للبناء العادي:
- انقر مرتين على: build-electron.bat

للبناء النهائي المحسن:
- انقر مرتين على: build-electron-final.bat

الملفات الناتجة:
- Windows: .exe و .msi في مجلد release\
- macOS: .dmg في مجلد release\
- Linux: .AppImage في مجلد release\

========================================
5. استكشاف الأخطاء
========================================

مشكلة: "npm command not found"
الحل: تأكد من تثبيت Node.js بشكل صحيح

مشكلة: "Failed to install dependencies"
الحل: 
1. تأكد من اتصال الإنترنت
2. امسح مجلد node_modules
3. امسح package-lock.json
4. اكتب: npm install

مشكلة: "Build failed"
الحل:
1. تأكد من وجود جميع الملفات
2. تحقق من مساحة القرص
3. أعد تشغيل الكمبيوتر

مشكلة: "Electron app won't start"
الحل:
1. تأكد من بناء التطبيق أولاً
2. تحقق من وجود مجلد dist\
3. جرب تشغيل: npm run electron
